3️⃣ DSA Questions (Expected for 3 YOE)

A. Arrays  ✅ 

* Find the maximum / minimum in an array
* Reverse an array
* Rotate array by K positions
* Find duplicate elements
* Two Sum
* Move zeros to end
* Kadane’s Algorithm (max subarray sum)
* Merge two sorted arrays


B. Strings  ✅ 

* Reverse a string
* Check palindrome
* Anagram check
* Longest common prefix
* Count character frequency
* First non-repeating character
* String compression

C. Hashing  ✅ 

* Two Sum using hashmap
* Find duplicates using hashmap
* Subarray with given sum
* Frequency-based problems


D. Linked List

* Reverse a linked list
* Detect loop (Floyd’s cycle)
* Find middle of linked list
* Merge two sorted linked lists
* Remove nth node from end


E. Stack & Queue

* Valid parentheses
* Implement stack using array
* Next greater element
* Queue using two stacks


F. Recursion & Backtracking (Basic)  ✅ 

* Fibonacci (recursive vs iterative)
* Factorial
* Power of number
* Generate all subsets (basic level)


H. Searching & Sorting  ✅ 

* Binary search
* Bubble, Selection, Insertion sort
* Time complexity comparison
