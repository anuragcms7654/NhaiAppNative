Microfrontend is an approach where a large frontend app is broken into smaller, independent pieces, just like microservices in backend.
-- Each team can build, deploy, and manage their part independently.

1. Module Federation (Webpack 5)
apps share code at runtime, Load components from another app dynamically
Example	- Cart app using Navbar from another app

-- Module Federation allows different frontend apps to share components and dependencies dynamically at runtime without rebuilding the entire app.

2. Single-SPA
-- Single-SPA helps integrate multiple frontend applications and manages which app should load based on routing.
